# Contributing to Caosmos UI 🖥️

Thank you for your interest in contributing to the Caosmos Visual Interface! This project provides the spatial visualization and interaction layer for the Caosmos engine.

## 🛠️ Development Setup

1. **Prerequisites**: 
   - **Node.js** (v20 or higher recommended).
   - **npm** or **pnpm**.
2. **Clone the repo**: `git clone https://github.com/alexpicode/caosmos-ui.git`
3. **Install Dependencies**: `npm install`
4. **Environment**: Copy `.env.example` to `.env` and configure the backend URL if necessary.
5. **Run Locally**: `npm run dev`

## 🎨 Tech Stack & Standards

- **React + Vite + TypeScript**.
- **CSS**: We use Vanilla CSS for maximum flexibility.
- **Components**: Keep components focused, reusable, and well-typed.
- **State**: Ensure synchronization with the backend simulation state is efficient.

## 🧪 Linting & Formatting

- Run the linter before committing: `npm run lint`
- Ensure there are no TypeScript errors: `npm run type-check` (if configured).

## 📨 How to Submit a Contribution

1. **Fork** the repository.
2. Create a **Feature Branch** (`git checkout -b feature/ui-improvement`).
3. **Commit** your changes following [Conventional Commits](https://www.conventionalcommits.org/).
4. **Push** to the branch (`git push origin feature/ui-improvement`).
5. Open a **Pull Request** against the `develop` branch.

## 💬 Community & Communication

If you have questions, feel free to reach out via [alexpicode@proton.me](mailto:alexpicode@proton.me) or open a GitHub Issue for discussion.
